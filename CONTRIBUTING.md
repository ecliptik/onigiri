# Contributing

Onigiri is a personal project, built for one household's tracking and
shared so others can read it, learn from it, and run their own builds.

- **Issues and discussions are welcome** — bug reports, questions, and
  ideas are all useful.
- **Pull requests are generally not accepted.** The app ships
  commercially under the maintainer's copyright while the repository is
  licensed [PolyForm Noncommercial 1.0.0](LICENSE); accepting outside
  code would require contributors to sign over broader rights than the
  license grants, and this project prefers to stay simple instead. Fork
  freely for your own noncommercial use.

## Name and icon

The **Onigiri** name and icon identify this project and its official
builds. The license covers the code — not the branding. If you
distribute a fork or a build of one, give it its own name and icon so
nobody mistakes it for the official app.
